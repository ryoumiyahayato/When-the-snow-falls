《雪降之时》魔法世界设定集（核心规则版）

一、世界本体与基本规则

1. 魔力的本质
魔力是一种可被生成、储存、转移、消耗与计量的基础资源。

其本质为通用能源体系，具备以下特性：
- 可自然存在于环境中
- 可被生物吸收与调度
- 可被法器储存与转换
- 可被标准化计量
- 可作为社会性资源使用

魔力不依赖人格、信仰或意志存在，本质为客观资源。

2. 魔力等级（核心系统）

魔力等级定义为：
单位时间内魔力通过人体的最大通量上限。

核心特性：
- 与魔力总量无关
- 与知识储备无关
- 与法术种类无关
- 只决定魔力“流动速度与处理上限”

等级结构：
A++：≈99（接近未知上限）
A+：96–99
A：≈95
B：≈85
C：≈75
F：<50

附加性质：
- 魔力等级决定术式构筑速度
- 魔力等级决定复杂术式承载能力
- 魔力等级不等于战斗力上限

3. 魔力储量（独立变量）

魔力储量与魔力等级完全独立。

定义：
个体或载体所能存储的魔力总量上限。

特性：
- 可自然积累
- 可外部注入
- 可通过法器承载
- 可被封存、转移或抽离
- 可作为资源流通与交换基础

结构关系：
- 魔力等级 = 单位时间流动能力
- 魔力储量 = 总容量上限

典型结构组合：
- 高等级 + 高储量
- 高等级 + 低储量
- 低等级 + 高储量
- 低等级 + 低储量

4. 魔法本质定义

魔法是：

魔力的结构化组织与输出技术。

构成要素：
- 魔力（基础资源）
- 术式结构（组织方式）
- 控制精度（执行能力）
- 环境变量（外部条件）

魔法体系整体属于：
可工程化、可复现的经验技术系统。

二、术式与结构体系

1. 术式段位体系

术式“段”表示结构复杂度与魔力组织层级。

结构层级：
低段：基础单体术式（光镖等）
中段：复合结构术式（球状闪电等）
高段：系统级术式（大结界、大规模术式）
十二段以上：实验级结构（不可稳定复现）

2. 法术演化机制（减魔体系）

所有现代法术均来源于历史演化：

高魔耗原始术式 → 结构解析 → 能效优化（减魔） → 模块化重构 → 标准化教学体系

结果：
- 术式逐渐工程化
- 高阶术式向低消耗版本转化
- 魔法学习体系趋向普及化

3. 魔法类型体系

元素系：火 / 水 / 风 / 土 / 光 / 电
空间系：传送 / 折叠 / 封锁 / 空间门 / 空间稳定
时间系：观测 / 加速 / 局部逆推 / 未来趋势推演
幻术系：感知信息干涉
结界系：局部区域魔力规则修改
炼金/构筑系：魔力→可持续物质结构
魔药系：魔力调制与生理适配体系
召唤系：契约结构与外部能量投射

4. 关键机制说明

结界：局部规则重写系统
反封印术：结构解析 + 权限重构 + 节点拆解
幻术：多感官信息干涉与误导

三、魔法使用者体系

1. 继承人
主流魔法使用者，通过学习掌握标准化体系

2. 维系者
极稀有规则继承者，继承对象为“规则”而非术式
多与时间/空间/因果相关
不可工程化复制

代表：凉宫凌体系

3. 掘墓人
异常反魔力个体
表现为术式结构不稳定与失败率上升
对复杂术式影响更强
